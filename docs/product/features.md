# Продуктовые фичи ИКС ВЕТ

## Концепция

Онлайн-платформа ветеринарной помощи. Владельцы питомцев получают консультации проверенных ветеринаров в чате — без очередей, без стресса для животного, в любое время суток.

**Две роли:** Владелец питомца | Ветеринар

---

## 1. Лендинг

- Hero-блок с CTA: "Найти врача" (→ дашборд владельца) и "Я ветеринар" (→ дашборд ветеринара)
- Социальное доказательство: аватары клиентов + "48 000+ владельцев доверяют нам"
- Блок статистики: среднее время ответа (12 мин), 1200+ врачей, 48 000+ консультаций
- Секция "Как это работает": 3 шага (Выберите врача → Напишите в чат → Получите помощь)
- Превью врачей: 3 карточки с рейтингом, статусом онлайн, специализацией
- Отзывы клиентов (pull-quote стиль)
- CTA-футер: "Начните сегодня"

---

## 2. Дашборд владельца

### 2.1 Приветствие
- Персонализированное приветствие с именем
- Кнопка "Найти врача"

### 2.2 Мои консультации
- Список предстоящих и завершённых консультаций
- Каждая запись: врач, питомец, дата, специализация, статус (Предстоит / Завершено)
- Клик → переход в чат

### 2.3 Мои питомцы
- Список питомцев с основной информацией (вид, порода, возраст, вес)
- Кнопка "+ Добавить" — форма нового питомца:
  - Кличка, вид (выпадающий список), порода
  - Возраст, вес, пол
  - Чекбокс "Стерилизован/кастрирован"
  - Текстовое поле "Особые отметки" (аллергии, хронические болезни)

### 2.4 Модальный профиль питомца (клик по карточке)
- Сетка характеристик: вид, порода, возраст, вес, пол, стерилизация
- Блок "Особые отметки" (выделен предупреждением — жёлтая плашка)
- История консультаций (дата, врач, проблема, результат)
- Кнопки: "Записать к врачу", "Документы"

---

## 3. Поиск ветеринара

### 3.1 Фильтры
- Текстовый поиск (имя или специализация)
- Фильтр по специализации (pills): Все, Терапевт, Дерматолог, Офтальмолог, Педиатр, Онколог
- Тогл "Только онлайн"

### 3.2 Список результатов
- Каждый врач: имя, специализация, рейтинг (звёзды), кол-во отзывов, опыт (лет), город, статус онлайн/офлайн, теги специализаций
- Кнопка "Записаться" у каждого

### 3.3 Пустое состояние
- "Никого не нашлось" + "Попробуйте изменить фильтры"

---

## 4. Профиль ветеринара

### 4.1 Информация о враче
- Аватар, ФИО, специализация, опыт, город
- Рейтинг + кол-во отзывов
- Статус онлайн
- Подробная биография
- Верификация диплома (бейдж)

### 4.2 Блок записи (sticky sidebar)
- Дата консультации
- Сетка слотов времени (09:00, 10:30, 12:00, 14:00, 15:30, 17:00)
- Кнопка "Записаться на [время]"
- Подтверждение записи (зелёная галочка + детали)
- Кнопка "Написать в чат"
- Возможность "Изменить время"

### 4.3 Отзывы
- Pull-quote стиль: текст отзыва, имя, питомец, рейтинг

---

## 5. Чат (консультация)

### 5.1 Сайдбар бесед
- Поиск по беседам
- Список бесед: имя, питомец, последнее сообщение, время, счётчик непрочитанных
- Индикатор онлайн у контактов

### 5.2 Основной чат
- Заголовок: аватар, имя, статус, питомец
- Сообщения: пузыри (свои справа, чужие слева), таймстемпы
- Прикрепление файлов (иконка скрепки)
- Прикрепление фото (иконка изображения)
- Превью прикреплённых файлов в сообщении (имя файла, размер)
- Индикатор "печатает..." (анимированные точки)
- Автоответ от врача (имитация в прототипе)
- Отправка по Enter, Shift+Enter для новой строки

---

## 6. Дашборд ветеринара

### 6.1 Заголовок
- Имя врача, бейдж "Диплом верифицирован", статус онлайн
- Кнопка "Открыть чаты"

### 6.2 Расписание на сегодня
- Таймлайн слотов: время, пациент, питомец, причина обращения
- Статусы: "Завершено" (приглушено), "Предстоит", свободные слоты

### 6.3 Новые запросы
- Карточки входящих запросов: имя владельца, питомец, описание проблемы, время подачи
- Бейдж "Срочно" (жёлтый) для срочных случаев
- Кнопки: "Принять" / "Отклонить"

### 6.4 Недавние пациенты
- Список: имя владельца, питомец, проблема, дата
- Клик → переход в чат

---

## Общие UX-паттерны

- Навигация: sticky header с логотипом, ссылками (зависят от роли), аватаром пользователя, кнопкой "Выйти"
- Hover-эффекты на интерактивных элементах (indent, подсветка)
- Анимации: fadeUp при загрузке, slideIn для элементов
- Цветовая кодировка статусов: зелёный (онлайн, завершено), терракотовый (предстоит, акцент), жёлтый (срочно, предупреждение)
